# Security Policy

## Reporting a vulnerability

Please report security issues **privately** — do **not** open a public issue.

Use GitHub's [private vulnerability reporting](https://docs.github.com/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)
on the affected repository (Security → Report a vulnerability). We aim to acknowledge within a few
days.

> Enable private vulnerability reporting in each repo's **Settings → Code security** (or org-wide)
> so the "Report a vulnerability" button appears.

## Data handling

Meowmoir keeps private user data in the user's own iCloud (CloudKit); the backend stores only
non-private operational data and never persists photos. Reports about data handling are welcome.
